Mobile apps and web based software for P2P delivery startup

About Project

KD is the product of Kundandeep Solutions Private Limited. It is a point-to-point delivery startup that was made to help people deliver anything anywhere. The requirements and architecture of the project were quite complex, involving 2 mobile Applications and an admin web dashboard. But HelioWeb team made every requirement possible from designing to deploying the final application. Few of the core features that we implemented are:
- Real time tracking of delivery partners through web sockets
- Completely autonomous daily salary and incentive calculator
- Shopkeeper app to place and track orders
- Customised account management within the dashboard
- Scaling the application to multiple cities with no changes in configuration
